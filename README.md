# 个人知识库

地址： [Code Guide](http://110.41.50.108/)

## 介绍

### 为什么选择 Docusaurus？

使用了很多的知识库工具，各有各的优劣，从语雀到思源笔记，再到现在的`docusaurus` ， 因为想随时随地的分享自己的知识库，并且更喜欢`React`一些，所以选择了这个工具。

### 有什么内容呢？

包含了各种各样的内容，但是绝大部份是计算机相关的知识，但是随着我写的文档越来越多，我发现了一个问题，就是写了很多`杂项`，就是突然来了性质，或者本身就是属于工具类型的内容，内容并不深，也不算多，无需太费笔墨。所以，针对这样的内容，我会把`未成型的`、`临时想到的`内容放到一个杂项中，而其他的内容打算使用下面的写作方式：

1. 在初次学习的时候，掌握的并不好只是入门学习，所以会记录相关的基础知识。
2. 当基础知识学习完毕之后，，而对于前面的内容就会简单的带过了，然后会加入自己的心得体会，即我自己提炼的框架，这部分即相对深度的内容了。
3. 当学习更深度的内容之后，对于简单部分的内容就会更少提及，而会加入更多的自己的心得体会，以及对其他领域的扩展，这部分即深度内容了。

## 运行指南

安装依赖

```bash
pnpm install
```

运行

```bash
pnpm run start
```

构建

```bash
pnpm run build
```
